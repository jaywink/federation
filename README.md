# Social-Federation

Python library to abstract social federation protocols.

Aims to first support Diaspora and then the W3C SocialWG ActivityPub protocol.

## Requirements

* Python 3.x

## Testing

Install requirements:

    pip install -r test-requirements.txt

Run tests:

    py.test
